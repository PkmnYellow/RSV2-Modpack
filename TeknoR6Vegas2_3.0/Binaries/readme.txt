==Support==
Flag #1:
- Can Increase Maximum Players to 16 in Terrorist Hunt
- Can Increase Maximum Players to 16 in Versus
- Can Increase Maximum Players to 16 in Story Mode
Flag #2:
- Can Increase Terrorists In Terrorist Hunt to certain count
Flag #3:
- Eyefinity Patch for 3 Monitors

==Usage==
1. Copy to your <Rainbow Six Vegas 2 game folder>\Binaries
2. Run from Console or use included BAT files.
3. Have fun!!!

==Caveats==
